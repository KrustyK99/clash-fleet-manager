Snapshot Collector

Paste one in-game account export at a time. The collector timestamps each paste immediately, resolves the account by player tag, and keeps a browser recovery draft until a successful import. Existing batch JSON files still work as a fallback.
Paste one snapshot
Copy the JSON from one Clash account, paste it here, then click Add Snapshot. Repeat for each account. The captured time is recorded when you add each snapshot, so timer end times are based on when the game export was taken rather than when the final import runs.
